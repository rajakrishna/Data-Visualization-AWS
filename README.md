# Data Visualization Website
Tools used: AWS Elastic Beanstalk, AWS RDS, AWS Redis, Python, Flask, MS SQL, D3.js, AWS EC 2
-	Developed a Flask web application where users would choose intervals or attributes in a dataset and the application will visualize the data in multiple charts
-	Implemented AWS Elastic Beanstalk to host the website, AWS RDS to upload the datasets and AWS Redis to cache the results. Data visualization was implemented by using D3.js
